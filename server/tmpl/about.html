Acerca de

- La navegación web o dentro de una aplicación para personas sin el sentido de la vista o con vista reducida es casi imposible. Por esa razón es necesario adaptar cada aplicación o pagina web que desarrollemos con accesibilidad disponible para ese tipo de personas y por esta razón decidimos desarrollar interfaces tanto móvil como de escritorio para personas que no pueden ubicar donde se encuentra el ratón o su selección Touch a la hora de navegar por un menú.


- Por esta razón, nos decidimos a crear un juego con interfaz para registro y login capaz de ser usable por gente con falta de vista o vista reducida, así llevando el entretenimiento a gente que no tiene a la mano este tipo de medios accesibles con sus diferentes capacidades.
