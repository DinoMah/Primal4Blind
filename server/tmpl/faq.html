PREGUNTAS FRECUENTES

Como jugar?

Para jugar no hay restricciones, solo necesitas un control o mando con el cual vas a navegar por el juego, y opcionalmente unos audifonos o auriculares por si deseas escuchar mejor todo a tu alrededor y tener una mejor experiencia


Por que es necesario usar un Control?

Es necesario el uso de un control o mando ya que el juego tambien esta hecho para personas que no tienen la posibilidad de saber manejar todas las teclas de un teclado, el control facilita el movimiento del juego y es mas accesible tanto para dichas personas con discapacidad visual como para las que no cuentan con ella.


En caso de tener alguna queja?

En caso de alguna incoformidad duda o sugerencia puedes notificarnos solicitando un formulario desde esta seccion ubicada en la parte inferior.
